# Portfolio

Questo progetto è stato generato usando [Angular CLI](https://github.com/angular/angular-cli) versione 20.3.8.

## Server di sviluppo

Per avviare un server di sviluppo locale, esegui:

```bash
ng serve
```

Una volta avviato il server, apri il browser e vai su `http://localhost:4200/`. L'applicazione si ricaricherà automaticamente quando modifichi i file sorgente.

## Generazione del codice

Angular CLI include potenti strumenti per generare codice. Per creare un nuovo componente, esegui:

```bash
ng generate component component-name
```

Per una lista completa di schematics disponibili (come `components`, `directives` o `pipes`), esegui:

```bash
ng generate --help
```

## Build

Per compilare il progetto esegui:

```bash
ng build
```

Questo compilerà il progetto e salverà gli artefatti nella cartella `dist/`. Di default, la build di produzione ottimizza l'applicazione per prestazioni e velocità.

## Eseguire i test unitari

Per eseguire i test unitari con il runner [Karma](https://karma-runner.github.io), usa il comando:

```bash
ng test
```

## Eseguire i test end-to-end

Per i test end-to-end (e2e), esegui:

```bash
ng e2e
```

Angular CLI non include di default un framework per i test end-to-end; puoi scegliere quello che preferisci.
